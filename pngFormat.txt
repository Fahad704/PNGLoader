8 byte header: 89 50 4E 47 0D 0A 1A 0A

Series of Chunks

Chunk:
	4 byte length
	4 byte type/name
	length byte data
	4 byte CRC checksum(network-byte-order CRC-32)


Critical chunks:
IHDR : 13 bytes data:			offset
	4 byte width				0
	4 byte height				4
	1 byte bits per channel		8
	1 byte color type			9
	1 byte compression method	10
	1 byte filter method		11
	1 byte interlace method		12

PLTE : Pallete information for indexed file

IDAT : DEFLATE compressed image data in multiple IDAT blocks , 
	->if there are multiple IDAT blocks then there must be nothing between them.
	->To make a single DEFLATE compressed data stream you need to concatenate all IDAT block data
